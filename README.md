# BeatMaker

A 16-step drum sequencer with 8 synthesized drum sounds, built entirely with the Web Audio API.

## Features

- **8 Synthesized Drum Tracks**: Kick, Snare, Hi-Hat, Open Hat, Clap, Tom, Rim, Crash â€” all generated in real-time via Web Audio oscillators and noise buffers
- **16-Step Sequencer Grid**: Click cells to toggle beats, click step numbers to jump playback position
- **Per-Track Controls**: Individual volume sliders, mute, and solo buttons for each track
- **Genre Presets**: Load pre-built patterns for Rock, Hip-Hop, House, Drum & Bass, Jazz Swing, Disco, Trap, and Reggae
- **Undo/Redo**: Full history support for pattern edits
- **Audio Export**: Render beats to WAV, MP3, or OGG with configurable loop count
- **Variable BPM**: Adjustable tempo from 60 to 180 BPM
